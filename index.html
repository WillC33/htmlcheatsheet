HTML Cheatsheet

Click on a card to copy the snippet

<html>

The root element that contains all other HTML elements.

<html>
  <head>
  </head>
  <body>
</html>

<head>

Contains meta-information about the document, such as the title and links to stylesheets.

<head>
  <title>Page Title</title>
</head>

<body>

Contains the visible content of the webpage.

<body>
  <h1>Hello, World!</h1>
</body>

<h1> to <h6>

Heading elements for defining headings with varying levels of importance.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

<p>

Defines a paragraph of text.

<p>This is a paragraph.</p>

<a>

Creates a hyperlink to another web page or resource.

<a href="https://www.example.com">Visit Example</a>

<img>

Embeds an image on the page.

<img src="image.jpg" alt="Description">

<ul> and <li>

Creates an unordered (bulleted) list with list items.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol> and <li>

Creates an ordered (numbered) list with list items.

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

<div>

A generic container for grouping and styling elements.

<div>Content goes here</div>

<span>

A generic inline container for styling a specific portion of text or inline elements.

<span style="color: red;">This is red text</span>

<a>

Creates a hyperlink to another web page or resource.

<a href="https://www.example.com">Visit Example</a>

<img>

Embeds an image on the page.

<img src="image.jpg" alt="Description">

<table>

Creates a table for organizing data into rows and columns.

<table>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
  <tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 2</td>
  </tr>
</table>

<form>

Creates a form for user input and data submission.

<form action="/submit" method="post">
  <input type="text" name="username" placeholder="Username">
  <input type="password" name="password" placeholder="Password">
  <input type="submit" value="Submit">
</form>

<input>

Creates an input field for various types of user input.

<input type="text" name="name" placeholder="Enter your name">
<input type="checkbox" name="subscribe" value="yes"> Subscribe to newsletter
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

<button>

Creates a clickable button element.

<button onclick="alert('Button clicked!')">Click me</button>
